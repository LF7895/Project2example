Project Description

Method 1 puts the p5 sketch into a div and assigns it an id of "sketch-container". This is done in the sketch-1.js file in the setup function. That div is styled using CSS (see the #sketch-conatiner attributes in the styles-1.css file. BTW, this page is not the cleanest implementation of flex-box, so please try to improve my code! Also notice the canvas height (fixed at 600) and the windowResized functions in the sketch files.
